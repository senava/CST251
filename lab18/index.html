Challenge

The challenge for today was to work with our partner and be able to create a responsive layout for our page. That meaning, having the page look good, no matter what size of screen you are looking at.

Problems

I did seem to run into a few problems when working with this lab. One of the struggles I was having was being albe to use the @media screens attribute in the CSS stylesheet. I was able to get some of my side-by-side elements to go to 100% width when shrinking the page, as well as having my flex boxes go back to block style, but when I shrink the page some more, some of my elements don't go the full 100%. I tried seeing which elements needed the @media screens attribute, but I wasn't able to figure it out.

Results

I gave it my ebst try and this is what I was able to come up with. When shrinking the page, until it reaches 600px, some of my elements that have the flex style to them will show as display. The images are able to scale down when resizing the page, but never bigger than their original size.
