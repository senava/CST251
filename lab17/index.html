Challenge

For this lab, we are working with our partners. Creating a layout by incorporating each of the CSS layout tools, giving our page a different look to it. Being able to sketch out a layout for our page and placing elements in different areas, giving them a unique layout to them. As well as having to create 2 different pages that show different techniques to them, like using the Float attribute and the Flexbox attribute.

Problems

For this first page, I didn't really run into any difficulties. It was pretty easy, since I anly had to add my content and leave everything as it is.
