Lab 19: CSS Frameworks

Challenge

Todays challenge for Lab 19 was to experiment with Bootstrap. As always, we had to work with our partners and try to experiment with Bootstrap and see how the various elements work. Along with creating the proper folders and index files, and adding some JavaScript files for this to work.

Problems

Some of the problems we ran into for this lab was figuring out how Bootstrap works and how to add it to our html. When first clicking on the website, everything looked so confusing, it was like a whole other language that we didn't understand. We soon were able to figure it out, after periods of time looking at w3schools.

Results

If done correctly and having done all of my styling correct, you should be able to see all of the content on this page. I experimented with a few other elements I found on w3school about Bootstrap and decided to give it a try. The navbar should have different dropdown elements that let you click on the links and take you to some of my past labs. I also added a few carousel throughout my page, having some sort of slideshow with the images I added. If you scrunch the site, the navbar should collapse, hiding all of the links and showing the three lines icon at the far right. One more thing, if you scroll down, you should still be able to see the navbar visible.
